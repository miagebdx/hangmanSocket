##Usage

- git : `git clone https://github.com/miagebdx/hangmanSocket` or [Download here](https://github.com/miagebdx/hangmanSocket/tarball/master)
- build app : `make all`
- launch server : `./server 8888`
- launch client : `./client localhost 8888`
